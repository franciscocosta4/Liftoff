Stack: Laravel ou Node.js, WebSockets, PostgreSQL, Web Audio API, React/Vue (opcional)

Objetivo: Criar sessões de estudo imersivas com mapa de assentos e áudio ambiente.

Funcionalidades principais:

Sessões de estudo com duração definida
Contagem decrescente até à “aterragem”
Mapa de lugares em tempo real
Sincronização via WebSockets
Opções de áudio ambiente
Presença social (ver utilizadores e lugares ocupados)
Possíveis melhorias:

Estatísticas de estudo
Streaks de sessões
Ranking de tempo estudado
Desafios técnicos:

Sincronização em tempo real
Gestão de temporizadores globais
Concurrência em escolhas de lugares
